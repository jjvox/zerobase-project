## JavaScript 과제 리스트

JavaScript 구현 연습 </br>

Zerobase FE School 2차로 제출 했던 JavaScript 과제 입니다.

- 과제이기 때문에 라이브러리를 사용하지 않고 모두 직접 구현 했습니다.

- 배포는 Vercel을 사용 했습니다.
